# Student Grade Calculator

A Java-based desktop application designed as part of the **Codsoft** internship tasks. This application calculates a student's total marks, average percentage, and assigns a corresponding grade based on the performance.

---

## 🚀 Features
* **Interactive UI:** Built with Java Swing for a clean and user-friendly interface.
* **Dynamic Calculation:** Automatically calculates Total Marks and Average Percentage.
* **Grade Assignment:** Assigns grades (A, B, C, etc.) accurately based on pre-defined grading criteria.
* **Custom Exception Handling:** Includes robust validation to handle incorrect mark entries (using a custom `MarkException`).
* **Database Export:** Saves student grade summaries into a local CSV file (`Grade_Master_Database.csv`).

---

## 🛠️ Tech Stack
* **Language:** Java
* **GUI Framework:** Java Swing / AWT
* **Tools:** VS Code / Git

---

## 📂 Project Structure
* `Main.java` - Entry point of the application.
* `GradeUI.java` - Handles the graphical user interface components and layouts.
* `Student.java` & `Subject.java` - Core models representing student data and subject marks.
* `MarkException.java` - Custom exception for custom input validations.
* `Grade_Master_Database.csv` - The spreadsheet database storing final results.
